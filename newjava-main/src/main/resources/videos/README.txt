# Videos Directory

This directory is used to store videos for lessons. 

## Video Format
- Use MP4 format for better compatibility
- Recommended resolution: 1280x720 (720p) or 1920x1080 (1080p)
- Recommended aspect ratio: 16:9

## Naming Convention
Use a naming convention like: `course_id_lesson_id.mp4`
Example: `course_12_lesson_3.mp4`

## Adding Videos
1. Place video files in this directory
2. Update the lesson's video URL in the database to point to the file

## Sample Videos
For testing purposes, you can use sample videos from open source repositories
or create short demo videos.
